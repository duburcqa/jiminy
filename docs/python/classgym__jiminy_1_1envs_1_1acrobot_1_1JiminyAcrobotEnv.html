Detailed Description

Implementation of a Gym goal-environment for the Acrobot which is using Jiminy Engine to perform physics computations and Gepetto-viewer for rendering.

It only changes the observation mechanism wrt the base class JiminyAcrobotGoalEnv. See its documentation for more information.
