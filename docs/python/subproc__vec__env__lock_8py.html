Worker for each subprocess of SubprocVecEnvLock.

It enables the use of a threading.Lock. It is the only difference with the implementation provided with the class SubprocVecEnv of Gym OpenAI.

Remarks
remote and parent_remote are returned by the method Pipe of multiprocessing. It is a pair of connection objects connected by a pipe which by default is duplex (two-way). See multiprocessing documentation for more information.

This is a hidden function that is not automatically imported using 'from gym_jiminy import *'.
