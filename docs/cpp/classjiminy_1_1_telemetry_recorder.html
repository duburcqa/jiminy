result_t jiminy::TelemetryRecorder::createNewChunk ( )
private

Create a new file to continue the recording.

Each chunk shall have a size defined by LARGE_LOG_SIZE_GB and shall be suffixed by an increasing natural number.

Returns
SUCCESS if successful, the corresponding telemetry error otherwise.
